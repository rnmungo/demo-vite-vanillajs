# Consigna para alumnos

## Objetivo del proyecto

Construir un **perfil profesional / CV web** usando:

- HTML semántico
- CSS propio
- Bootstrap
- JavaScript

La idea es que el proyecto crezca durante la cursada y después te quede como base de portfolio.

## Recomendación de trabajo

No trabajes solamente sobre un fork si querés que tus commits cuenten mejor en GitHub.

Lo más conveniente es:

1. crear un repositorio propio a partir de esta base
2. clonar tu repositorio
3. trabajar y hacer commits sobre tu rama principal

## Archivos principales

- `index.html` → estructura del sitio
- `src/styles/main.css` → estilos propios
- `src/main.js` → comportamiento con JavaScript

## Clase 1

### Meta

Construir la estructura visual de tu perfil profesional.

### Tenés que trabajar sobre:

- `header`
- navegación
- `#presentacion`
- `#sobre-mi`
- `#habilidades`
- `#proyectos`
- `footer`

### Se espera que practiques

- etiquetas semánticas
- jerarquía de títulos
- organización de contenido
- estilos con CSS
- layout con Bootstrap

### Entrega esperada al final de la clase

Una primera versión funcional de tu sitio, aunque no esté terminada.

## Tarea para después de clase 1

- completar textos reales
- mejorar diseño y espaciados
- revisar responsive
- completar la sección de proyectos
- ordenar navegación interna

## Clase 2

### Meta

Agregar comportamiento con JavaScript a tu sitio.

### Tenés que trabajar sobre:

- `src/main.js`
- la sección `#contacto`

### Ideas de funcionalidades

- validar formulario
- mostrar mensajes de error o éxito
- mostrar u ocultar contenido
- agregar un botón interactivo
- resaltar una sección al hacer click

## Criterios de evaluación sugeridos

- estructura semántica clara
- contenido coherente y prolijo
- uso razonable de Bootstrap
- estilos propios presentes
- JavaScript aplicado con sentido
- commits frecuentes y progreso visible

## Publicación

Cuando el proyecto esté listo:

1. hacé push a GitHub
2. publicalo con GitHub Pages
3. compartí la URL final
